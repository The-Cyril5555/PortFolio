# Images de Peintures Classiques pour le Portfolio

Ce dossier contient les images de peintures classiques utilisées comme arrière-plans pour les cartes de projets.

## Images à télécharger

Téléchargez 5 peintures classiques et renommez-les comme suit :

### 1. `proflex.jpg` - Pour le projet Proflex (Intérim Logistique)
- **Suggestion** : Peinture baroque avec des personnages dynamiques
- **Sources** :
  - Unsplash : https://unsplash.com/s/photos/baroque-painting
  - Met Museum : https://www.metmuseum.org/art/collection/search#!?q=baroque
  - Exemple : Une scène de commerce ou d'activité

### 2. `sqwk.jpg` - Pour le projet SQWK (E-commerce)
- **Suggestion** : Nature morte classique ou scène de marché
- **Sources** :
  - Unsplash : https://unsplash.com/s/photos/classical-still-life
  - Exemple : Nature morte avec fruits, objets

### 3. `wiki.jpg` - Pour l'application Wiki
- **Suggestion** : Scène de bibliothèque ou d'érudits
- **Sources** :
  - Unsplash : https://unsplash.com/s/photos/classical-library-painting
  - Exemple : Philosophes, livres, scène d'étude

### 4. `sdis.jpg` - Pour SDIS 29 (Technicien IT)
- **Suggestion** : Peinture avec thème technique/architectural
- **Sources** :
  - Unsplash : https://unsplash.com/s/photos/renaissance-architecture
  - Exemple : Scène d'atelier, architecture

### 5. `cma.jpg` - Pour CMA29 (Support IT)
- **Suggestion** : Scène de bureau ou administrative classique
- **Sources** :
  - Unsplash : https://unsplash.com/s/photos/classical-painting-office
  - Exemple : Portrait, scène d'intérieur

## Format recommandé
- **Dimensions** : Minimum 800x600px (idéal : 1200x800px)
- **Format** : JPG ou PNG
- **Poids** : < 500KB si possible (optimisé pour le web)

## Après téléchargement
Une fois les 5 images téléchargées et placées dans ce dossier avec les noms ci-dessus,
le fichier `projects.data.ts` sera automatiquement mis à jour pour utiliser ces images locales.
